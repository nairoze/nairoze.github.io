Steps to Create GitHub Pages:
1. Create a new repository.
2. Push Code and files in the GitHub repository.
3. Under the repository name, go to Settings.
4. Select 'Pages' from the left-hand sidebar menu.
5. Under the 'Build and deplyoment' section, make sure that te source is set to "Deploy from a branch.
6. Select the main branch then click Save.
7. The live URL will appear.
